# Task Manager (Kotlin) 

A simple command-line app made in Kotlin to manage your daily tasks — add, list, complete, or remove them easily.

## What it does
-  Add a task (title + description)
-  List all tasks (with ID, status, and description)
-  Mark a task as completed
-  Remove a task

## How to run it
1. Open the project in IntelliJ (or any Kotlin IDE)
2. Run the `main.kt` file
3. Follow the menu in the console — pretty straightforward!

- Kotlin
